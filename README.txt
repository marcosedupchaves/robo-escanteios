ROBO DE ESCANTEIOS - VERSÃO V20

1. Suba este repositório no GitHub.
2. No Railway:
   - Clone/GitHub: selecione seu repositório.
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `python main.py`
   - Environment Variables:
     * API_FOOTBALL_KEY
     * TELEGRAM_TOKEN
     * CHAT_ID
3. Deploy automático a cada push.
4. O bot envia RELATÓRIO AUTOMÁTICO a cada 10 minutos 
   e responde aos comandos /start, /ajuda e /odds.
5. Para atualizar código: push no GitHub → Railway faz redeploy.

Qualquer dúvida, me avise!
